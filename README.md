# Support-Page
Support page for the "Right Resume - Right Now" browser extension

## About the Extension

Right Resume - Right Now is a Chrome extension that automatically generates tailored resumes and cover letters for job applications. Simply visit a job posting and let the extension craft a customized application document.

## Features

- **Smart Job Analysis**: Extracts key requirements from job postings
- **Tailored Documents**: Generates resumes and cover letters matched to each job
- **One-Click Generation**: Create documents with a single click
- **Subscription Plans**: Free tier with premium options for unlimited generations

## Getting Started

### Installation
1. Install from Chrome Web Store
2. Click the extension icon in your browser
3. Sign up or log in with your email
4. Start on any job board (LinkedIn, Indeed, Glassdoor, etc.)

### How to Use
1. Navigate to a job posting
2. Click the Right Resume extension icon
3. Review the job details
4. Click "Generate Resume" or "Generate Cover Letter"
5. Download and customize if needed

## Troubleshooting

### Extension Not Working?
- Refresh the job posting page
- Make sure you're on a supported job board
- Check that JavaScript is enabled in Chrome
- Try signing out and back in

### Generation Failed?
- Check your internet connection
- Verify you haven't exceeded your monthly limit (free tier: 5/month)
- Try refreshing the page and retry

### Can't Sign In?
- Check your email and password
- Clear Chrome cookies for this site
- Try in an incognito window
- Refresh the page

## Privacy & Security

- We never store your personal documents
- Job posting content is analyzed in real-time only
- Your resume data is encrypted
- We comply with Chrome Web Store privacy policies

## Account Management

### View Your Subscription
1. Click the extension icon
2. Go to "Settings" or "Account"
3. View your current plan and usage

### Cancel Subscription
1. Go to account settings
2. Click "Manage Subscription"
3. Select "Cancel"

### Delete Your Account
Contact support with your email address for account deletion.
